# 学生信息管理系统 python628

## 项目概述

学生信息管理系统是一款基于B/S架构的开发项目，采用Python Django作为后端开发框架，Vue3作为前端技术栈，实现了一套完整的前后端分离的系统。

## 技术栈

- 架构：B/S
- 系统环境：Windows/Mac
- 开发环境：
 - Pycharm
 - Python3.0
 - Django
 - Mysql
 - NodeJs
- 技术栈：
 - Python
 - Django
 - Mysql
 - Vue3

## 功能模块

### 用户端

- 首页
- 商品列表
- 商品分类
- 商品搜索
- 购物车
- 商品详情
- 购买支付（对接沙箱模拟支付）
- 个人中心

### 管理端

- 管理员登陆
- 商品信息管理
- 商品类型管理
- 订单信息管理
- 用户管理

## 运行环境

- 后端：Python3.0 + Django + Mysql
- 前端：Vue3
